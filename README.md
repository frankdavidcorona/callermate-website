# Caller Mate Website

Marketing website for Caller Mate platform using React.

# Instalation

```sh
$ git clone git@github.com:samsystems/callermate-website.git
$ cd callermate-website
$ yarn
$ yarn prod
```

# Deploy
```sh
$ npm i -g pm2
$ cd callermate-website
$ pm2 start app.js -i 0 --name "CallerMate.com"
```
